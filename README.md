# Data Analysis in R

This project analyzes national welfare and GDP data using R.

## Skills Used
- Data cleaning
- Subsetting datasets
- Loops and conditional statements
- Matrix operations
- Merging datasets
  
## Data
The datasets used in this project include national welfare and GDP data.  
Some files are large and may not preview directly in GitHub, but they can be downloaded and used with the provided R script.
